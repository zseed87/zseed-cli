## 简单的脚手架
### 安装

```
# install it globally
$ npm install -g zseed-cli

# or yarn
$ yarn global add zseed-cli
```
### 使用

```
$ zseed create <project-name> [-f|--force]
```
### 参数

```
-f, --force: Overwrite if the target exists
```
